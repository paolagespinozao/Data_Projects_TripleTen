# Data_Projects_TripleTen
A collection of projects I developed while completing the Business Intelligence Analytics program at TripleTen.


| Project number | Title | Tools | Description |
| :-----------: | ----------- |----------- |----------- |
| 1 | Music Platform Performance Analysis | Excel/Google Sheets | Evaluated Fresh Beats’ promotion strategies, user engagement, and genre performance. Summarized insights and delivered recommendations to improve artist exposure, increase user conversion, and enhance platform engagement. |
| 2 | Manhattan Airbnb Market Analysis | Excel/Google Sheets | Evaluated Manhattan Airbnb data to determine the best neighborhoods and property sizes for vacation rentals. Used pivot tables to analyze demand, calculated revenue for top listings, and produced recommendations for profitable real estate investment.|
| 3 | End-to-End SQL Analysis for Venture Capital Decision-Making | SQL | Analyzed VentureInsight’s database of startups, funding rounds, acquisitions, investors, and people. Covered startup survival, sector funding, acquisition trends, social media influencer detection, global funding distribution, fund activity levels, and education-based success analysis. |
| 4 | E-Commerce Conversion & Cohort Analysis | Excel/Google Sheets |Built an end-to-end e-commerce conversion funnel and cohort analysis using spreadsheet techniques. Identified unique users per funnel stage, created purchase-based cohorts, calculated monthly retention, and summarized insights in a polished executive report. |
| 5 | SuperStore Profitability & Operations Analysis | Tableau |Analyzed SuperStore data to uncover profit trends, loss areas, effective advertising opportunities, and high-return-risk products and customers, using visualizations to support strategic recommendations. |
| 6 | Root Cause Analysis of SuperStore’s High Return Rates | Tableau |Analyzed SuperStore orders and returns to identify key drivers of high return rates. Created visualizations by product, customer, region, and time, and built an interactive Tableau dashboard to monitor patterns and support data-driven strategies to reduce returns. |
| 7 | Shopify App Marketplace Analysis | Power BI | Analyzed Shopify apps and reviews in Power BI, creating visualizations to explore app counts, review trends, and developer responsiveness, highlighting key factors driving app success. |
